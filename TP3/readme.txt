performance.py est le script permettant d'effectuer les tests de performances.

*****Il est necessaire d'avoir Python 3.X d'installer afin d'utiliser le script.*****

Usage : python ./performance.py <IP Address of web server> 8000
*Ici, 8000 represente le port des serveurs.

Ainsi, pour un serveur (ou load balancer) ayant l'adresse 132.207.12.102, le script s'execute comme suit :

python .\performance.py 132.207.12.102 8000
